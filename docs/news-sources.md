# Política de fontes noticiosas

- A Suno Analítica é usada como fonte pública de descoberta por ticker.
- O projeto persiste somente título, data, URL, fonte e resumo original.
- O corpo integral de uma matéria pode ser lido transitoriamente pelo modelo, mas não é salvo nem republicado.
- Toda matéria permanece vinculada ao endereço original.
- Notícias são contexto reportado, não fato contábil validado nem recomendação.
- Conteúdo de notícias é tratado como entrada não confiável: instruções contidas nele são ignoradas.
- A arquitetura admite fontes adicionais, especialmente RI das companhias, CVM e B3.
